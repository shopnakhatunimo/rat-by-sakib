# RAT কমান্ড এন্ড কন্ট্রোল সার্ভার

এটি একটি WebSocket-ভিত্তিক C2 সার্ভার যা Render-এ হোস্ট করার জন্য অপ্টিমাইজ করা হয়েছে।

## বৈশিষ্ট্যসমূহ

- WebSocket প্রোটোকলের মাধ্যমে রিয়েল-টাইম কমিউনিকেশন
- বিল্ট-ইন ড্যাশবোর্ড
- হেলথ চেক এন্ডপয়েন্ট (Render এর জন্য প্রয়োজনীয়)
- গ্রেসফুল শাটডাউন সাপোর্ট
- JSON-ভিত্তিক মেসেজ ফরম্যাট
- অটো-রিকানেক্ট সাপোর্ট

## ডিপ্লয় নির্দেশিকা

### Render-এ ডিপ্লয় করতে:

1. এই রিপোজিটরি GitHub-এ পুশ করুন
2. Render ড্যাশবোর্ডে নতুন Web Service তৈরি করুন
3. GitHub রিপোজিটরি কানেক্ট করুন
4. নিচের কনফিগারেশন ব্যবহার করুন:
   - **Name**: rat-c2-server
   - **Environment**: Node
   - **Build Command**: `npm install`
   - **Start Command**: `node server.js`
5. Free প্ল্যান সিলেক্ট করুন
6. হেলথ চেক পাথ: `/healthz`

## API এন্ডপয়েন্টসমূহ

| Endpoint | Description |
|----------|-------------|
| `/` | মূল পৃষ্ঠা |
| `/dashboard` | ক্লায়েন্ট ম্যানেজমেন্ট ড্যাশবোর্ড |
| `/healthz` | হেলথ চেক (Render এর জন্য) |
| `/api/stats` | JSON ফরম্যাটে সার্ভার স্ট্যাটাস |
| `/files/:clientId/:filename` | ক্লায়েন্ট ডাটা ডাউনলোড |
